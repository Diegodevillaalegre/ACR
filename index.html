1) WPI: Índice de dolor generalizado (0–19) — última semana

Cómo usar WPI

Marque cada zona con dolor durante la última semana. Suma de 0 a 19. Para "dolor generalizado" se requieren ≥4 de 5 regiones con dolor. Pecho y abdomen no cuentan para esa regla.

2) SSS: Severidad de síntomas (0–12)

Cómo usar SSS

Asigne un puntaje exacto a cada dominio: 0 = ausente, 1 = leve, 2 = moderado, 3 = severo. Seleccione el número explícito para evitar ambigüedad.
